# 🏴‍☠️ Log Piece

Help Luffy become the King of the Pirates — one commit at a time.

## What is this?

Log Piece is a VS Code extension that turns your git commits into a One Piece story. Every commit you make progresses Luffy's journey from Foosha Village all the way to Elbaf, the island of giants.

## Features

- 🌊 291 story milestones covering the entire journey from East Blue to Elbaf
- 🎨 Original pixel art for the East Blue milestones (more on the way)
- 📖 Ship's Log sidebar showing your current saga, arc, and progress
- ⚓ Commit counter tracking your journey
- 🏴‍☠️ Notification on every milestone reached
- 📊 Arc progress bar showing how far you are within the current arc

## How it works

Just install the extension and make commits as normal. The Ship's Log panel in your Explorer sidebar updates automatically as your commit count grows.
